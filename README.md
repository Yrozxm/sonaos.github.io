# SonaOS Website

SonaOS is a handcrafted, human-centric operating system designed for focus and stability. This website serves as the digital presence for the project, showcasing its philosophy, features, and upcoming releases.

## Features

- **Obsidian-Grade Design**: Deep blacks, high contrast, and a minimalist aesthetic.
- **Human-Centric Philosophy**: Technology that serves the user, not the other way around.
- **Hardware Respect**: Optimized for performance on both new and old machines.
- **Stable Foundation**: Built upon Debian 13 "Trixie".
- **Blog**: Insights on digital craftsmanship, focus, and the SonaOS journey.
